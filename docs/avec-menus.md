# AVEC — menus & sous-menus

Références : [ICI — guide AVEC](https://www.cocoainitiative.org/fr/centre-de-ressources/ressources/guide-dimplementation-associations-villageoises-depargne-et-de), [50MAWSP RDC](https://share.google/aqDKkozFFvF3p7LzB), modèle McBuleli (`docs/avec-model.md`).

## Principes AVEC → écrans

| Pratique AVEC | Où dans l’app |
|---------------|----------------|
| Cycle d’épargne (9–12 mois) | **Vue** — jauge de cycle |
| Réunions régulières (ex. hebdo) | **Réunion** — parts 1–5, cotisation |
| Fonds de solidarité | **Vue** + **Réunion** (montant social) |
| Caisse commune transparente | **Vue** + **Caisse** + **Rapports** |
| Adhésion / anciens & nouveaux membres | **Membres** |
| Crédit interne / versements | **Caisse** — payouts (prêts à venir) |
| Dialogue, preuves, conflits | **Dialogue** |
| Traçabilité (carnet, audit) | **Rapports** |

## 6 onglets

### 1. Vue (aperçu)
**But :** décision rapide — synthèse visuelle, peu de texte.

- Caisse totale, objectif de cycle, progression %
- Graphique : cotisations par période
- Répartition membres (actifs / en attente)
- Top épargnants (parts / USDT)
- Alertes : abonnement, réunion, demandes en attente
- Raccourcis vers Réunion / Membres

### 2. Réunion
**But :** action de la réunion — acheter des parts.

- Choix 1–5 parts, montant dû, payer depuis portefeuille USDT
- Rappel : valeur de part fixée pour le cycle

### 3. Membres
**But :** gestion des adhérents.

- Liste approuvés (épargne, parts, rôle)
- File d’attente : approuver / refuser
- Rôles admin / co-admin (lien réglages)

### 4. Caisse
**But :** sorties de trésorerie (crédit, récompenses).

- Versements aux membres (managers)
- Solde disponible

### 5. Dialogue
**But :** communication du groupe (réunion, preuves, médiation).

- Fil de messages
- Pièces jointes / preuves

### 6. Rapports
**But :** transparence comptable.

- Journal des écritures USDT
- Preuves partagées
- Journal d’audit (managers)
